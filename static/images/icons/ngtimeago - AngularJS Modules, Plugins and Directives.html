ng-timeago

AngularJS module for simple time ago on static data. No directives here, just a filters.

Quick start

bower install ng-timeago new version bower install ng-timeago0.0.2

or alternatively download and include ngtimeago.js after angular.min.js. Add the ngtimeago module as a dependency when creating your app, e.g. var app = angular.module('myApp', ['ngTimeago']);` NO NEED TO INJECT in controller directly use in html.
